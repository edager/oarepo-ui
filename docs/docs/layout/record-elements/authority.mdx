# Authority

An element that renders a personal or organizational authority information.

```yaml
component: 'authority'
```

:::

## Properties

| Name                   | Description                                                           |
| ---------------------- | --------------------------------------------------------------------- |
| `fullName`             | full name of a person or an organization                              |
| `role`                 | role name of a person or an organization                              |
| `authorityIdentifiers` | list of [identifier](./authority-identifier)s of an authority         |
| `nameType`             | type of authority, can be either `Personal` or `Organizational`       |
| `affiliations`         | list of authority affiliations                                        |
| `wrapperComponent`     | _CDB_ description of component wrapping the authority element parts   |
| `fullNameComponent`    | _CDB_ description of component rendering a full name part             |
| `identifierComponent`  | _CDB_ description of component rendering an authority identifier part |
| `roleComponent`        | _CDB_ description of component rendering role info part               |

## Data

Uses any data referenced by the `dataField` as authority props:

```jsx
const {fullName, authorityIdentifiers, role} = referencedData

<Authority
  fullName={fullName}
  authorityIdentifiers={authorityIdentifiers}
  role={role} ... />
```
