## Properties

<!-- TODO: describe any properties accepted by an element -->

| Name                 | Description                                                      |
| -------------------- | ---------------------------------------------------------------- |
| fullName             | full name of a person or an organization                         |
| role                 | role name of a person or an organization                         |
| authorityIdentifiers | list of [identifier](./authority-identifier)s of an authority    |
| fullNameComponent    | _CDB_ description of component rendering a full name             |
| identifierComponent  | _CDB_ description of component rendering an authority identifier |
| roleComponent        | _CDB_ description of component rendering role info               |

## Data

Uses any data objects referenced by the `dataField` as authority props:

```jsx
const {fullName, authorityIdentifiers, role} = referencedData

<Authority
  fullName={fullName}
  authorityIdentifiers={authorityIdentifiers}
  role={role} ... />
```
