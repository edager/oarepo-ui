# Defense status

Shows the status of work's (typically thesis) defense.

```yaml
component: 'defense-status'
```

## Properties

| Name                   | Description                                                         |
| ---------------------- | ------------------------------------------------------------------- |
| `defended`             | whether the work was successfully defended or not                   |
| `dateDefended`         | date of the thesis defence                                          |
| `defendedComponent`    | _CDB_ description of component showing that work was defended       |
| `notDefendedComponent` | _CDB_ description of component showing that work was _NOT_ defended |

## Data

Uses any data referenced by the `dataField` as defense status props:

```jsx
const {defended, dateDefended} = referencedData

<DefenseStatus
  defended={defended}
  dateDefended={dateDefended}
  ... />
```
