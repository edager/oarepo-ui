# Result header

Renders a clickable header link for records search result item

```yaml
component: 'result-header'
```

- **Jinja2**: üèó

:::

## Properties

| Name         | Description                                                                                                           |
| ------------ | --------------------------------------------------------------------------------------------------------------------- |
| `title`      | Main title of a record                                                                                                |
| `links`      | An object containing record links                                                                                     |
| `linksField` | Path reference to links object stored in record's data                                                                |
| `*`          | any other properties passed on to [Header](https://semantic-ui.com/elements/header.html#/definition) wrapping element |

:::tip Additional Docs

- [Semantic-UI Header](https://semantic-ui.com/elements/header.html#/definition)
- [Semantic-UI React Header](https://react.semantic-ui.com/elements/header/)

:::

## Data

- uses record's data referenced by the `dataField` as Header title
- uses record's data referenced by the `linksField` to create a link to record's detail page

```jsx
const {title} = referencedDataFromDataField
const {self: selfLink} = referencedDataFromLinksField

<Header
  title={title}
  href={selfLink}
  ... />
```
