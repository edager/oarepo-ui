# Authority identifier

An element that renders an identifier badge of person or organization.

```yaml
component: 'authority-identifier'
```

## Properties

| Name         | Description              |
| ------------ | ------------------------ |
| `identifier` | identifier value         |
| `scheme`     | scheme of the identifier |

## Data

Uses any data referenced by the `dataField` as authority identifier props:

```jsx
const {identifier, scheme} = resolvedData

<AuthorityIdentifier identifier={identifier} scheme={scheme}
```
