# Custom icon

A customizable [Semantic-UI Icon](https://semantic-ui.com/elements/icon.html#/definition) element.

A standard Semantic-UI Icon element supports rendering of provided icons only. This element
adds options to provide custom graphics to be rendered as an icon image or to render icons
from provided custom icon set.

```yaml
component: 'custom-icon'
```

## Properties

| Name      | Description                                                                                                                                                                                                                                                                  |
| --------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `name`    | icon name, referring either to [Semantic-UI Icon](https://semantic-ui.com/elements/icon.html#/definition) name, or name from provided `iconSet`                                                                                                                              |
| `iconSet` | map of custom icons this element can render. Accepts keys as icon name, values as either [Semantic-UI Icon](https://semantic-ui.com/elements/icon.html#/definition) name or an object with [Semantic-UI Image](https://semantic-ui.com/elements/image.html#image) properties |
| `*`       | any other props passed to icon element                                                                                                                                                                                                                                       |

## Data

Uses any data referenced by the `dataPath` as icon name:

```jsx
<CustomIcon name={referencedData} {/*...*/} />
```

## Example

### Semantic-UI icon

<ExampleTabs
  layout={[
    {
      component: 'custom-icon',
      name: 'thumbs up',
      color: 'green',
      size: 'large',
    },
  ]}
/>

### Custom icon from an icon set

<ExampleTabs
  layout={[
    {
      component: 'custom-icon',
      name: 'custom',
      iconSet: {
        custom: {
          src: 'https://semantic-ui.com//images/wireframe/image.png',
        },
      },
    },
  ]}
/>
