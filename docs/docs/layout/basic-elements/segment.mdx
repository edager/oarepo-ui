# Segment

A [Semantic-UI Segment](https://semantic-ui.com/elements/segment.html#/definition) element.

```yaml
component: 'segment'
```

:::note Implementation state:

- **Jinja2**: [✅](https://github.com/oarepo/oarepo-ui/blob/master/src/oarepo_ui/templates/oarepo_ui/components/basic/segment.html.jinja2)

:::

## Data

Uses any data referenced by the `dataField` as a Segment `children`:

```jsx
<Segment {/*...*/}>{childrenFromData}</Segment>
```
