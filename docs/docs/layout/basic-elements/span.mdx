# Span

A stylable text span element.

```yaml
component: 'span'
```

:::note Implementation state:

- **Jinja2**: [✅](https://github.com/oarepo/oarepo-ui/blob/master/src/oarepo_ui/templates/oarepo_ui/components/basic/span.html.jinja2)
- **React**: [✅](https://github.com/oarepo/react-generative-ui/blob/master/src/components/basic/Span.tsx)

:::

## Properties

| Name | Description                                     |
| ---- | ----------------------------------------------- |
| `*`  | any props are passed to the `span` HTML element |

:::

## Data

Uses any data referenced by the `dataField` as a span `children`:

```jsx
<span {/*...*/}>{childrenFromData}</span>
```
