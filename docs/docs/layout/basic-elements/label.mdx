## Properties

| Name    | Description                                       |
| ------- | ------------------------------------------------- |
| content | contents of a label                               |
| `*`     | any props are passed to Semantic-UI Label element |

## Data

Uses any data referenced by the `dataField` as Label `content`:

```jsx
<Label content={contentFromData} {/*...*/} />
```
