# Header

A [Semantic-UI Header](https://semantic-ui.com/elements/header.html#/definition) element.

```yaml
component: 'header'
```

:::note Implementation state:

- **Jinja2**: [✅](https://github.com/oarepo/oarepo-ui/blob/master/src/oarepo_ui/templates/oarepo_ui/components/basic/header.html.jinja2)
- **React**: [✅](https://github.com/oarepo/react-generative-ui/blob/master/src/components/basic/Header.tsx)

:::

## Data

Uses any data referenced by the `dataField` as header `content`:

```jsx
<Header content={contentFromData} {/*...*/} />
```
