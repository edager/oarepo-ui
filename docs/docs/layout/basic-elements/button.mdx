# Button

A [Semantic-UI Button](https://semantic-ui.com/elements/button.html#/definition) element.

```yaml
component: 'button'
```

:::note Implementation state:

- **Jinja2**: [✅](https://github.com/oarepo/oarepo-ui/blob/master/src/oarepo_ui/templates/oarepo_ui/components/basic/button.html.jinja2)
- **React**: [✅](https://github.com/oarepo/react-generative-ui/blob/master/src/components/basic/Button.tsx)

:::

## Data

Uses any data referenced by the `dataField` as button element `children`:

```jsx
<Button {/*...*/}>
  {children}
</Button>
```
