# List

A [Semantic-UI List](https://semantic-ui.com/elements/list.html#/definition) element
grouping its contents into a list of items.

```yaml
component: 'list'
```

## Properties

| Name         | Description                                                  |
| ------------ | ------------------------------------------------------------ |
| `item`       | _CDB_ description of how individual items should be rendered |
| `items`      | list of items, either strings or objects with item props     |
| `horizontal` | list should be rendered horizontally                         |
| `*`          | any additional props passed to the wrapping list element     |

:::tip Additional Docs

- [Semantic-UI List](https://semantic-ui.com/elements/list.html#/definition)
- [Semantic-UI React List](https://react.semantic-ui.com/elements/list/)

:::

## Data

Uses any data referenced by the `dataField` as list `items` data:

```jsx
<List items={referencedData} ... />
```

<ExampleTabs

### Horizontal data list

<ExampleTabs
  noResult
