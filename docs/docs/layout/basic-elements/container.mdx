# Container

A [Semantic-UI Container](https://semantic-ui.com/elements/container.html#/definition) element.

```yaml
component: 'container'
```

:::note Implementation state:

- **Jinja2**: [✅](https://github.com/oarepo/oarepo-ui/blob/master/src/oarepo_ui/templates/oarepo_ui/components/basic/container.html.jinja2)

:::

## Data

Uses any data referenced by the `dataField` as container `children`:

```jsx
<Container {/*...*/}>
  {children}
</Container>
```

## Example

### Simple container

<ExampleTabs
