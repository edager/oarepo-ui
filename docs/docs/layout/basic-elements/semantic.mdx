# Semantic-UI elements

A raw [Semantic-UI](https://semantic-ui.com/) elements can be used
given by its name. Any props from _CDB_ are passed directly to
the element.

If no Semantic-UI element could be found by the name, a plain HTML element is used.

```yaml
component: '<semantic-ui-element-name>'
```

## Properties

| Name | Description                 |
| ---- | --------------------------- |
| `*`  | any supported element props |

:::tip Additional Docs

- [Semantic-UI](https://semantic-ui.com/)
- [Semantic-UI React](https://react.semantic-ui.com/)

:::

## Data

Renders any data referenced by the `data` path as:

```jsx
<semantic-ui-element-name {/*...*/}>{referencedData}</semantic-ui-element-name>
```
