# Link

An element used to render simple hyperlinks.

```yaml
component: 'link'
```

:::note Implementation state:

- **Jinja2**: [✅](https://github.com/oarepo/oarepo-ui/blob/master/src/oarepo_ui/templates/oarepo_ui/components/basic/link.html.jinja2)
- **React**: [✅](https://github.com/oarepo/react-generative-ui/blob/master/src/components/basic/Link.tsx)

:::

## Properties

| Name | Description                                        |
| ---- | -------------------------------------------------- |
| href | link target path / URL                             |
| `*`  | any props are passed to the `<a></a>` html element |

## Data

Uses any data referenced by the `dataField` as a `href` link target:

```jsx
<a href={targetFromData} {/*...*/} />
```
