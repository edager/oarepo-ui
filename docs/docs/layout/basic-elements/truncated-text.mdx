# Truncated text

Renders the input text truncated to a given amount of lines, with a possibility to expand to full view.

```yaml
component: 'truncated-text'
```

## Properties

<!-- TODO: describe any properties accepted by an element -->

| Name           | Description                                     |
| -------------- | ----------------------------------------------- |
| `lines`        | number of lines displayed in truncated state    |
| `ellipsis`     | ellipsis character (default: `‚Ä¶`)               |
| `text`         | main text content to be rendered truncated      |
| `expandToggle` | props passed to 'Show more\|less' toggle button |
| `*`            | any extra props passed to wrapping element      |

:::tip Additional Docs

- [React-Text-Truncate](https://www.npmjs.com/package/react-text-truncate)

:::

## Data

Renders any data referenced by the `dataPath` path as the `text` prop:

```jsx
<TruncatedText text={referencedData} {/*...*/} />
```
