# Separator

Used to separate blocks of content in a row.

```yaml
component: 'separator'
```

:::note Implementation state:

- **Jinja2**: [✅](https://github.com/oarepo/oarepo-ui/blob/master/src/oarepo_ui/templates/oarepo_ui/components/basic/separator.html.jinja2)
- **React**: [✅](https://github.com/oarepo/react-generative-ui/blob/master/src/components/basic/Separator.tsx)

:::

## Properties

| Name     | Description                                                     |
| -------- | --------------------------------------------------------------- |
| `double` | whether separator should be rendered doubled                    |
| `color`  | specify a foreground color of a separator                       |
| `*`      | any other properties and styles passed to wrapping span element |

## Data

:::info Data rendering

Doesn't directly render any data referenced by the `dataField`.

:::
