# Column

A [Semantic-UI Grid Column](https://semantic-ui.com/collections/grid.html#columns) container element that
renders its contents in a responsive column.

```yaml
component: 'column'
```

## Properties

| Name    | Description                                |
| ------- | ------------------------------------------ |
| `items` | list of _CDB_ descriptions of column items |
| `*`     | any extra props applied to column element  |

:::tip Additional Docs

- [Semantic-UI Grid Columns](https://semantic-ui.com/collections/grid.html#columns)
- [Semantic-UI React Grid.Column](https://react.semantic-ui.com/collections/grid/)

:::

## Data

:::info Data rendering

Doesn't directly render any data referenced by the `dataPath`.
