# Row

A layout element that places its items in a flexbox row.
Items can be separated by an optional separator.

```yaml
component: 'list'
```

:::note Implementation state:

- **Jinja2**: üèó

:::

## Properties

<!-- TODO: describe any properties accepted by an element -->

| Name        | Description                                                |
| ----------- | ---------------------------------------------------------- |
| `items`     | list of _CDB_ items to be rendered in a row                |
| `separator` | either a string or a _CDB_ used to separate rendered items |
| `*`         | any other properties passed to the flexbox row             |

:::tip Additional Docs

- [Semantic-UI Grid Row](https://semantic-ui.com/collections/grid.html#grouping)
- [Semantic-UI React Grid.Row](https://react.semantic-ui.com/collections/grid/)
- [CSS Flexbox](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Flexible_Box_Layout/Basic_Concepts_of_Flexbox)

:::
