# Row

A layout element that places its items in a grid row width
customizable columns.

```yaml
component: 'row'
```

:::note Implementation state:

- **Jinja2**: üèó
- **React**: [‚úÖ](https://github.com/oarepo/react-generative-ui/blob/master/src/components/structural/Row.tsx)

:::

## Properties

| Name            | Description                                                                                                                                                 |
| --------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `columns`       | list of _CDB_ items to be rendered in a row. Assumes [Column](column) definitions by default, any other component is wrapped in a [Column](column) element. |
| `columnsPerRow` | (default: `equal`) number of columns rendered in the GridRow                                                                                                |
| `children`      | children elements passed directly to the Grid Row wrapping element                                                                                          |
| `*`             | any other properties passed to the Grid Row                                                                                                                 |

:::tip Additional Docs

- [Semantic-UI Grid Row](https://semantic-ui.com/collections/grid.html#grouping)
- [Semantic-UI React Grid.Row](https://react.semantic-ui.com/collections/grid/)

:::
