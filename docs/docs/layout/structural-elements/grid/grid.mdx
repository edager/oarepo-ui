# Grid

A [Semantic-UI Grid](https://semantic-ui.com/collections/grid.html#/definition) container element that
renders its contents in a responsive columns and/or rows.

```yaml
component: 'grid'
```

:::note Implementation state:

- **Jinja2**: [✅](https://github.com/oarepo/oarepo-ui/blob/invenio-3.5-jinja/oarepo_ui/templates/oarepo_ui/components/components.html.jinja2#L2)
- **React**: [✅](https://github.com/oarepo/react-generative-ui/blob/master/src/components/structural/Grid.tsx)

:::

## Properties

### Grid properties

| Name            | Description                                                          |
| --------------- | -------------------------------------------------------------------- |
| `className`     | space-separated list of CSS classes applied to grid                  |
| `columnsPerRow` | (default: `equal`) number of columns rendered per each row in a grid |
| `columns`       | list of _CDB_ descriptions of [Columns](grid/column#properties)      |
| `rows`          | list of _CDB_ descriptions of [Grid Row](grid/row#properties) items  |
| `container`     | boolean - whether the grid should act as a responsive container      |
| `*`             | any extra props passed down                                          |

:::tip Additional Docs

- [Semantic-UI Grid](https://semantic-ui.com/collections/grid.html#/definition)
- [Semantic-UI React Grid](https://react.semantic-ui.com/collections/grid/)

:::

## Data

:::info Data rendering

Doesn't directly render any data referenced by the `dataField`.

:::
